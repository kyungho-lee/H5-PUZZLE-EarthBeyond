## 핵심 인사이트

### 1. 초보 × 적극/열심 = 광고 폭탄 문제
- 중급 이하 + CONTINUE 많이 쓸수록 광고가 급증
- 초보가 CONTINUE를 90%로 누르면 **중앙 160회**, 극단값 556회
- 별 수입(1⭐/일)이 CONTINUE 비용(5→10→20→40)을 따라잡지 못함
- **결론: 초보에게 CONTINUE를 자주 권하는 게 역효과**

### 2. 중급 × 소극 = 건강한 구간
- 중급 + 소극(CONTINUE 30%) → **광고 중앙 7회, 평균 일수 14일**
- 2주 만에 테마 하나 완성, 광고 부담 낮음 → 유저 입장에서 허용 가능

### 3. 별 수입이 결정적
- 중급+3⭐/일 vs 중급+2⭐/일: 광고 수는 비슷(5 vs 7)하나 테마 완성 일수는 동일
  → **step 도달 능력이 병목**이지 별 수입이 아님 (step 도달 확률 모델 한계)
- 고수 + 5⭐/일: **소극이면 광고 1회**로 완성 가능 → 과금 없는 플레이 실질 가능

### 4. Daily 미플레이 시 (별 수입 0)
- 중급 + 소극 → 광고 중앙 11회 (daily 있을 때 7회 vs 11회, 차이 4회)
- 큰 차이는 아니지만 daily를 유도하는 효과 있음

### 5. step 9(size 512)가 주요 병목
- 중급 기준 step 9(512) 최초 달성까지 중앙 **2일**, 75%는 3일
- 반면 step 10(1024) 달성은 중앙 **7일**, 75%는 12일
- → **마지막 두 단계가 전체 소요 시간의 절반 이상** 차지

## 밸런스 권고

### A. 지금 당장 문제없는 범위
- **중급 이상 + 소극/적극**: 테마 완성 9~14일, 광고 7~28회 → 광고 수익 적정
- **고수**: 완성 3~5일, 광고 1~8회 → 조기 완성은 다음 테마 해금 동기로 긍정적

### B. 개선 검토 필요한 범위
- **초보 + 적극/열심**: 광고 120~240회, 완성 25~35일 → UX 악화 위험
  - 개선안 1: **초보(step 6=64 미달 시) CONTINUE 최대 2회로 자동 제한**
  - 개선안 2: **1회차 CONTINUE는 항상 무료(광고 없이)** 제공 → 첫 경험 개선

### C. 별 수입 경제 점검
- CONTINUE 비용 누적(5+10+20+40=75⭐/세션)에 비해 daily 수입(1~5⭐/일)이 적음
- 초보가 2회 CONTINUE/일 한다면 75⭐ × 2회 필요 → 75일치 daily 별이 하루에 소모
- **별 수입 채널 다양화** 필요: 테마 완성 보상(⭐20) 외 추가 수단 고려

### D. THEME_COMPLETE_REWARD 효과
- 테마 완성 시 ⭐20 지급: 다음 테마 CONTINUE 4회분 스타 확보
- 이 보상 덕분에 테마 연속 플레이 시 체감 광고가 줄어드는 효과 있음
