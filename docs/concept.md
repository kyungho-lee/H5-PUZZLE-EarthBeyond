# Earth & Beyond — 게임 컨셉 & 피치

## 한 줄 요약

> **"머지 한 번이 수백만 년의 시간을 앞당긴다 — 원시 지구에서 우주의 끝까지"**

---

## 코어 루프

Age of 2048 류의 merge-cascade 구조에 **우주 연대기 시대 컬렉션**을 얹은 게임.
각 시대(Era)는 11단계 이미지팩으로 구성되며, 1024 머지 달성 시 해당 시대가 완성되고 다음 시대가 해금된다.

```
[타일 머지] → [step 승급] → [1024 달성] → [시대 완성] → [다음 Era 해금]
                                             ↓
                                    컬렉션 도감에 기록
```

### 3가지 플레이 모드

| 모드 | 목적 | 특징 |
|---|---|---|
| **Daily** | 경쟁 / 수익화 채널 | 하루 3회, 스코어 리더보드, 광고 리워드 |
| **Practice** | 진입장벽 제거 | 무제한 재시도, 기록 없음 |
| **Collection** | 스토리·장기 리텐션 | Era 순서대로 진행, 도감 수집 |

## 차별화 포인트

1. **큐레이션된 우주 사진 모티브 아트**
   - 허블, JWST, Voyager, Cassini, New Horizons 실사 이미지를 아트 소스로 사용
   - 단순 숫자/도형 타일이 아닌, 실제 천문학 장면을 게임 타일로

2. **시대별 고유 스킨**
   - 각 Era마다 보드 배경(board-bg) + 슬롯 배경(slot-bg01/02)이 다름
   - Era 1: 용암/마그마 오션 톤 / Era 2: 고대 문명 석재 톤 / Era 3: 심우주 칠흑+별빛

3. **우주 연대기 교육 콘텐츠**
   - 게임 자체가 138억 년의 우주 역사를 순서대로 체험하는 구조
   - 각 step에 실제 천문학/역사 배경 설명 제공 가능

4. **핸드큐레이션 77장 이미지**
   - Era 1~3 총 33장 + 배경/슬롯 이미지
   - Playgama 리젝 대응 근거: 자동 생성이 아닌 독자적 큐레이션 + 시대 스토리 구조

## 수익화 연계

- Daily 모드 리트라이: 광고 시청으로 재도전권 획득
- Era 해금: 별(Star) 재화 소모 또는 자연 진행
- 리더보드: Firebase Realtime DB + Playgama 스코어 API 이중 저장

---

## 기술 스택 요약

- 프레임워크: Vanilla JS (의존성 없음, H5 번들 단일 파일 배포 가능)
- 에셋: SVG placeholder → WebP 정식 전환 (ext 교체만으로 전환)
- 플랫폼 타겟: Playgama (1차), CrazyGames (2차)
- 스토리지: Firebase RTDB (리더보드, 컬렉션 진행도), Playgama Bridge SDK (스코어/광고)
